Sobre la privacidad acá

Estás usando una instancia hosteada de Anonimal. Tu texto se envía a este servidor para procesarse y no se almacena — pero, a diferencia del modo local, el dato sí sale de tu equipo hacia este servidor.

Para privacidad total —que el dato nunca salga de tu máquina—, auto-hosteá Anonimal: corrélo en tu propia computadora o servidor. Es open source.
